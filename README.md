# Learn docopt

I'll be tinkering around with docopt to see if that is easier for me to handle
than argparse.

## Install

```bash
sudo apt install python3.7
sudo pip install pipenv
pipenv install
```

## Use

```
pipenv shell
./hello.py -h
```
